金鸡独立？经济独立！

发表于 2021-12-20 | 分类于 随笔 | 阅读次数:

最近开始记账，包括每一笔流水，甚至一元钱都会记下来。记账就是很令人开心，因为这可是我一直梦想的经济独立呀。

小时候，幼儿园放学了，我和其他幼儿园小朋友一起等家长来接。等呀等，只剩老师陪我一起等了。老师问我，你家长怎么还不来呢？是呀，为什么呢？我只是低着头不说话。

再长大一些，我升入了小学，但还是要家长接送。周末的午后，妈妈载我到爷爷奶奶家。相对的两个房间，一间是烟雾缭绕的麻将室，充满大人癫狂的吼叫；另一间是卧室，安静到充满了死亡的气息。没有别的小朋友，有时候有书籍，偶尔有纸笔。什么都没有时我就站在窗边，一个人盯着太阳落下去。有时候盯着盯着，心里一酸，就开始哭泣。我把自己裹在窗帘后面哭，又怕被发现，又渴望被发现。呼，一次都没被发现过。哭完了还是没办法离开，我开始思考。我需要一件供我随意支配的交通工具，我需要钱，我需要经济独立。

于是，在连“经济”是什么意思都不清楚的年纪，我告诉自己我要经济独立。

一切为了有趣

发表于 2021-11-30 | 分类于 杂谈 | 阅读次数:

起源是豆瓣上有个女权帖子骂某些动森玩家在自制衣服时绘制大胸，其言辞义愤填膺。下面有条评论说，虽然自己也很气，但是想想自己也画过猛男比基尼就骂不出口了。我看到这条评论笑得要死。

觉得当下的男权女权就像是两个打架的小男孩和小女孩，小男孩脸被指甲抓破，小女孩头发被扯得乱糟糟。小男孩和小女孩总是想要对方的玩具，你要是当了真，撸起袖子帮忙干架，那可真的是傻乎乎。

男人和女人总是有无数的误解，渴望对方而又看不透对方因而相互物化。矛盾引发冲突，冲突中诞生悲喜交加的戏剧。女权太严肃了，一想到女权就想起浓郁的带着幽兰色芳香的悲哀。而男性的悲哀是另一种颜色，赤红色的焦灼难耐的伤痛。历史将枷锁套在男人女人身上，解救他们的也应当是那只无形大手，是变幻的风云。用扫把砸男生的脑袋这种事，我猜是没有意义的。

把生活的最高目的解释为有趣的话，一切便会顺利成章许多。

排序

发表于 2021-04-25 | 分类于 笔记 | 阅读次数:

一般都是用C++自带的sort排序，但基础的排序算法总是应该掌握的。

基础排序算法

选择排序： 模拟人类的手动排序，n轮遍历，每次挑出最小的元素放在最前面。时间复杂度$O(n^2)$，空间复杂度O(1)。

冒泡排序：不断比较相邻元素的大小，外层控制排序的轮数，内层如果是从前向后遍历比较的话，数组尾端的元素最先到位。时间复杂度$O(n^2)$，空间复杂度O(1)。

插入排序：不断比较相邻元素的大小，外层控制排序的轮数，内层如果是从前向后遍历比较的话，数组尾端的元素最先到位。时间复杂度$O(n^2)$，空间复杂度O(1)。

快速排序：取轴分割，小的在左，大的在右。分治，两边各自递归调用函数。时间复杂度$O(nlogn)$，空间复杂度O(1)。

归并排序：先分割到不能再分，然后再合并前半段和后半段（两有序数组并为一数组）。注意，需要再建一个空数组用于归并过程中存值。时间复杂度$O(nlogn)$，空间复杂度O(n)。
