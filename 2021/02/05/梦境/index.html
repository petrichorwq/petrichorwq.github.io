梦境

发表于 2021-02-05 | 分类于 随笔 | 阅读次数:

大概自去年，开始以文字的方式记录梦境。

我在梦里聚会畅谈、探寻废弃屋子、走进圣洁的大厅、邀请喜欢的男生到家里吃虫子、见到去世的亲人或仰慕的作家。也不总是愉快的，有时也会在狭窄走廊和成群的敌人拼刺刀、走进最肮脏恐怖的卫生间（去年看《猜火车》时，那间全英国最肮脏的厕所简直是我梦境的复现，不过我梦境可能更糟一点，我猜是不会有马桶的）、上厕所上到一半发现是公开场合、被讹几十万、和朋友闹掰。当然更多的是混乱的，茫茫然地东奔西顾，看着光怪陆离的陌生世界。

在整理记忆和备忘录时，我都有点不敢相信零零碎碎拼起来竟能成为如此精彩的一本集邮册。终日宅在家的平平无奇理科女，在午夜钟声敲响第十二下时，趁着夜色乘着名为Dream的鹰头马身有翼兽，经历着不比《哈利波特》逊色的冒险。

梦境中细节被模糊，到处泛着珠光色的雾气。在这样的朦胧中，我同自己多得数不清的念头和解，以荒诞的情节、折断的冲突或对往事的覆写。一年或两年前，看《残雪文学回忆录》，有一篇文章叫《不可能的戏》。“我”一遍又一遍追赶着早已逝去的大弟，一遍又一遍上演不可能的戏。当初是在梦里，之后是在文字里。

今天开始读伍尔夫的《奥兰多》。第二章有这样一段话：

在昏睡中，那些使人备受折磨的记忆，令人终身沮丧的往事，都被一只黑色的翅膀一抹而尽，它将苦涩擦去，再镀上黄金，即便是最丑陋、最卑贱的事，也被装饰得熠熠生辉。抑或是死亡的指尖不时轻触生命之烦忧，以免我们被这烦忧撕扯得粉身碎骨？也许我们生来就得日复一日一点一滴地浅尝死亡的滋味，方能生存下去？

苦涩被擦去，流淌着的是黄金。在梦境里，我忘记我曾在心理老师面前哭到卫生纸球堆成小山，用破碎的字句诉说我的后悔。因为我爷爷就在桌子对面坐着，穿着橙色衬衫，外面套着粉色毛衣，给我们斟饮料，笑得整排上牙露出来。

# 梦境 # 伍尔夫 # 奥兰多 # 残雪
