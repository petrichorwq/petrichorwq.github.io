排序

发表于 2021-04-25 | 分类于 笔记 | 阅读次数:

一般都是用C++自带的sort排序，但基础的排序算法总是应该掌握的。

基础排序算法

选择排序： 模拟人类的手动排序，n轮遍历，每次挑出最小的元素放在最前面。时间复杂度$O(n^2)$，空间复杂度O(1)。

冒泡排序：不断比较相邻元素的大小，外层控制排序的轮数，内层如果是从前向后遍历比较的话，数组尾端的元素最先到位。时间复杂度$O(n^2)$，空间复杂度O(1)。

插入排序：不断比较相邻元素的大小，外层控制排序的轮数，内层如果是从前向后遍历比较的话，数组尾端的元素最先到位。时间复杂度$O(n^2)$，空间复杂度O(1)。

快速排序：取轴分割，小的在左，大的在右。分治，两边各自递归调用函数。时间复杂度$O(nlogn)$，空间复杂度O(1)。

归并排序：先分割到不能再分，然后再合并前半段和后半段（两有序数组并为一数组）。注意，需要再建一个空数组用于归并过程中存值。时间复杂度$O(nlogn)$，空间复杂度O(n)。

#include<cstdio>

const int N = 8;
int arr[N] = {7,1,3,4,5,9,4,2};


void selectSort(int* arr, int N){
    for(int i=0; i<N-1; i++){
        int min = i;
        for(int j=i+1; j<N; j++){
            if(arr[min] > arr[j])
                min = j;
        }
        if(min != i)
            swap(arr[min], arr[i]);
    }
}

void insertSort(int* arr, int N){
    for(int i=1; i<N; i++){
        int pos = i-1;
        int value = arr[i];
        while(arr[pos] > value){
            arr[pos+1] = arr[pos];
            pos--;
        }
        arr[pos+1] = value;
    }
}

void quickSort(int* arr, int start, int end){
    // [s, e)
    if(start+1 >= end){
        return;
    }
    int l=start, r=end-1;
    int pivot = arr[l];
    while(l<r){
        while(l<r && arr[r]>pivot)
            r--;
        arr[l] = arr[r];
        while(l<r && arr[l]<=pivot)
            l++;
        arr[r] = arr[l];
    }
    arr[l] = pivot;
    quickSort(arr, start, l);
    quickSort(arr, l+1, end);
    return;
}

void mergeSort(int* arr, int l, int r, int* temp){
    // [l, r)
    if(l+1 >= r)
        return;
    int m = l+(r-l)/2;
    mergeSort(arr, l, m, temp);
    mergeSort(arr, m, r, temp);
    int p=l, q=m, i=l;
    while(p<m || q<r){
        if(q>=r || (p<m && arr[p]<arr[q]))
            temp[i++] = arr[p++];
        else
            temp[i++] = arr[q++];
    }
    for(int j=0; j<r; j++){
        arr[j] = temp[j];
    }
}
